standardisation of experimental data in a few steps
At first a vessel in which the reaction took place and its volume has to be defined. This can be for example an eppendorf tube or an Erlenmeyer flask.
1. vessels & volumes
The enzyme of the reaction has to be defined. It can be entered manually or entries from BRENDA can be taken. The concentration and other important descriptors like the amino acid sequence have to be defined by the user.
2. biocatalyst
In the next step the reactants of the reaction catalyzed by the enzyme have to be defined. Existing reactions can be taken from Strenda-DB or the reaction is defined manually. The number of substrates and products can be chosen individually. Concentration, purity and other important attributes can be defined. It is also possible to add new attributes, that are important for that specific experiment.
3. reactants
Now additional metadata has to be given. This includes the used buffer and its concentration. Also the temperature has to be defined here. It is also possible to add new, important attributes for example the absence or existance of a light soure.
4. reaction conditions
To complete the experiment, the experimental data or measurement data can be uploaded. An excel template is given, which allows an easy upload of the data. It is also possible to upload different measurement series. A plot of the data is given instantly and can be downloaded.
5. experimental data
After everything was entered the whole experiment can be uploaded to zenodo. It is also possible to create an EnzymeML file or a PDF report. If you have additional experiments, that differ by only one parameter from the original experiment, you can add these easily under "additional experiments" and don't have to define the whole reaction again.
